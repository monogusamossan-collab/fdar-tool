# FDAR・SOAP 自動記録メーカー

訪問リハビリ・訪問看護向け  
音声入力対応の記録作成支援ツールです。

---

## 🔹 このツールでできること

- FDAR形式での記録作成
- SOAP形式での記録作成
- 両形式の切り替え対応
- 音声入力での下書き作成
- ワンタップでコピー可能

---

## 🔹 特徴

✅ 移動中でも入力できる  
✅ スマホ対応  
✅ 個人情報は保存しない設計  
✅ インストール不要（URLだけ）

---

## 🔹 利用上の注意

※ 氏名・住所・電話番号などの個人情報は入力しないでください。
